
# Chess

A simple chess game written in C++ using the SDL2 library

You can run it by executing the Chess.exe in /bin or compile it yourself by using the Makefile in /bin. All necessary libraries are included.

```terminal
  cd bin
  mingw32-make
  .\Chess.exe
```
